Build_own_blog

Jul 13, 2023
在 Windows 中安装 hugo 🔗 在 PowerShell 中键入 :: choco install hugo-extended 安装完成后 hugo version 可以查看 hugo 的版本即为安装成功. 使用hugo 新建初始博客模块 🔗 键入 > hugo new site blog 在本地文件夹内新建 blog 文件(可自定义文件名),但是 hugo 需要默认的样式文件，下载后放在指定的 themes 文件夹中 git clone git@github.com:nodejh/hugo-theme-mini.git themes/mini 可以看到 themes 文件夹中有 mini 文件夹，即下好的样式文件 开始运行 🔗 hugo server -t mini –buildDrafts 运行后根据上面生成的 mini 文件编译网站,mini 根据以上 新建第一篇文章 🔗 hugo new post/build_own_blog.md 运行后网站就会出现 build_own_blog.md ，文件格式为markdown,可以修改下看看 编译静态文件 🔗 hugo –theme=mini –baseURL=“https://doubtduck.github.io/" –buildDrafts
